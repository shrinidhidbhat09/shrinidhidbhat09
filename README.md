## 🛠 Things I've Built

### Real-time Video Communication System

A system that handles **live video calls between users**.

Interesting parts:
- token based session management
- retry handling when connections fail
- keeping state consistent between clients

Tech stack included **Django, PostgreSQL and Agora**.

---

### BlinkEasy

A small computer vision service that detects **blink rate from video input**.

The idea was simple:

> If people stare at screens too long, their blink rate drops.

So the system detects it and warns them.

Stack:
MediaPipe + Flask + Firebase

---

### Movie Watchlist System

A simple but clean **data-driven application**.

Users can:

- manage preferences

The focus here was **clean state handling and structured data flow.**
